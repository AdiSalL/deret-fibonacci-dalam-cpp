apa itu fibonacci ?, 
fibonnaci merupakan sebuah deret angka yang diperoleh dari menjumlahkan dua angka sebelumnya.
contohnya seperti: 
0 1 akan menjadi 1
1 1 (satu atas masuk kesini) akan menjadi 2
1 2 akan menjadi 3
2 3 akan menjadi 5
